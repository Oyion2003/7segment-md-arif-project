# Design and Simulation of a 7-Segment Logic Display for “MD ARIF”

## Overview
This project presents the design and simulation of a combinational logic circuit to display the characters:

**M, d, A, r, I, F**

using a **7-segment display**.

The project was implemented in two methods:

1. **Direct Boolean Functions (Canonical SOP)**
2. **Karnaugh Map Simplified Boolean Functions**

## Tools Used
- NI Multisim
- Boolean Algebra
- Karnaugh Map (K-map)
- Common Anode 7-Segment Display

## Project Contents
- Full assignment report
- Direct function simulation
- K-map simplified simulation
- Truth table
- K-map derivation
- Output screenshots
